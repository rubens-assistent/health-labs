## 🎭 Interactions & Animations

### Page Transitions

| Transition | Duration | Easing |
|------------|----------|--------|
| Page enter | 300ms | ease-out |
| Page leave | 200ms | ease-in |
| Card hover | 150ms | ease-out |
| Modal open | 200ms | ease-out |
| Dropdown | 100ms | ease-out |

### Hover Effects

**Startup Card:**
```css
.card:hover {
  transform: scale(1.02);
  border-color: #3B82F6;
  box-shadow: 0 0 20px rgba(59, 130, 246, 0.15);
}
```

**Links:**
```css
.link:hover {
  color: #FAFAFA;
  text-decoration: underline;
  text-underline-offset: 4px;
}
```

### Scroll Animations

- **Hero text**: Fade in from bottom
- **Cards**: Staggered fade in
- **Stats**: Count up animation
- **Footer**: Fade in when visible

## 📱 Responsive Breakpoints

| Breakpoint | Width | Changes |
|------------|-------|---------|
| **Mobile** | < 640px | 1 column grid, hamburger menu |
| **Tablet** | 640-1024px | 2 column grid, compact header |
| **Desktop** | > 1024px | 4 column grid, full header |
| **Wide** | > 1280px | Larger typography, more spacing |
